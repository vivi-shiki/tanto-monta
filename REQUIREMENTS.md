# Tanto Monta — RTT 模组需求列表

> 将 tantomonta 前端原型移植为 Rally the Troops 服务器模组

## 项目来源

tantomonta 是一个基于 React + Vite 的前端原型，实现了《Tanto Monta》桌游的部分功能。
本项目 (tanto-monta) 将其移植为 RTT 服务器标准模组，使用纯 JavaScript 实现。

## 现有进度总结（tantomonta 原型）

### 已完成
- 回合/阶段状态机（抽牌→外交→春季→行动→冬季→婚姻→胜利），含跳过逻辑
- 行动阶段脉冲轮替（四个势力交替行动，pass 追踪，全部 pass 后结束）
- 区域盒编辑器（创建/拖拽/连接/编辑区域属性）
- 约 51 个区域的坐标和邻接数据
- 计数器/标记资源加载与棋盘放置
- 卡牌面板、手牌管理（抽牌/弃牌/使用）
- 招募动作（征召兵 1OP / 正规军 2OP / 骑兵 3OP）
- 陆上移动（1OP，堆叠管理，将领容量加成）
- localStorage 持久化

### 未完成
- 区域命名、类型元数据、VP 值
- 控制标记和归属系统
- 通信线 (LOC) 校验
- 场景初始部署（1470 / 1490 开局）
- 完整卡组数据和强制事件
- 战斗系统（野战/攻城/海战）
- 拦截、避战、撤退
- 外交系统（宣战/同盟/次要势力）
- 冬季维护（返回驻地/补员/损耗/瘟疫）
- 婚姻与王室事件
- 胜利判定
- 多人网络同步
- 测试覆盖

## 资源清单（从 tantomonta 复用）

- **地图**：`public/tm/main.jpg`（伊比利亚半岛底图）
- **计数器**：`public/counter/` 下约 100+ 个单位/将领/舰船 PNG
  - 势力：france, castille, portugal, nasrid, nafrica, aragon, beltranejos, catalonia 等
  - 类型：mil(征召兵), reg(正规军), cav(骑兵), art(炮兵), nao(远洋船), galley(桨帆船), corsair(海盗)
  - 将领：各势力 2-4 名历史将领
- **卡牌**：`public/card/` 下 131 张卡牌 PNG
  - event_01-64：主事件牌
  - event2_01-54：次事件牌
  - home_01-13：Home 卡
- **区域数据**：`public/data/region-boxes-*.json`
